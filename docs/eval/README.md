# H-CCX Benchmark Files

- `samples/` contains the single-part prompt pool.
- `multipart/` contains the multi-part prompt pool.
- `ccx_50/` contains runnable CalculiX kits for H-CCX evaluation.

The prompt pool has 466 authored JSON briefs: 318 single-part and 148
multi-part. The curated evaluation scope used by the runner scripts is 50
cases: 20 single-part and 30 multi-part.
